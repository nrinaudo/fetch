  60. def map[B](f: (A) ⇒ B): Request[B]

    Applies the specified transformation to the request's eventual response.

    Applies the specified transformation to the request's eventual response.

    Application developers should be wary of a common pitfall: when working with responses that contain instances of Response.Entity, they should always clean these up, either by reading their content (transforming it to something else or calling Response.Entity.empty) or explicitly ignoring them (by calling Response.Entity.ignore).

    This is a common source of issues when mapping error statuses to exceptions: each connection will be kept alive until the remote host decides it has timed out.
